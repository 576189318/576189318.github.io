hexo+github博客搭建

这几天用github搭建了hexo博客现在写篇文章试一下顺便讲一下自己的步骤和遇到的问题。

##一：准备

  1. 给电脑配置Node.js环境和git（网上和多教程，请自行查阅）



  2. 申请自己的GitHub账号。

##二：本地搭建

  1. 本地新建一个文件夹来存放博客文件（比如blog）


  2. 右击该文件，选中 Git Bash here


  3. 安装hexo

  4. 初始化博客

  5. 安装依赖包

  6. 生成静态网页

  7. 开启hexo服务端口（Hexo 会监视文件变动并自动更新，您无须重启服务器。）

如果安装成功会提示 INFO Start processing

此时打开浏览器输入 http://localhost:4000/就可以预览博客欢迎页面

###完成这些工作当然不是为了在本地查看，下面部署到我们的GitHub上去

  1. 打开文件_config.yml，编辑网站配置
  2. 把第 6 行的 title 改成你想要的名字
  3. 第九行author项为作者名
  4. 把最后一行的 type 改成 type: git
  5. 在最后一行后面新增一行，左边与 type 平齐，加上一行 repo: 仓库地址 请建立一个仓库,名为:GitHub帐户名.github.io,否则导致创建失败，地址为仓库文件的下载地址（即Clone or Download 选项提供的地址）

  6. 执行命令

  7. 将.deploy文件部署到GitHub

  8. 此时将所用仓库的GitHub Pages 的功能打开，就可以通过访问http://GitHub用户名.github.io浏览你的博客

###到这里hexo通过GitHub搭建结束，之后可以自己修改博客文件来发布自己的博客了。
