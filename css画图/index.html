css画图

看了一些简单的css画图代码，才发现css竟然还能这样用，css还用这种功能。

让我们简单了解一下：

css画图实际上也就是通过div的一些属性，通过css设置div的样式来改变css的形状和颜色。从而做出各种各样的图案。

  • 首先最简单的正方形


    • #square {
    width: 100px;
    height: 100px;
    background: red;
    }
  • 再来个圆


    • #circle {
    width: 100px;
    height: 100px;
    background: red;
    -moz-border-radius: 50px;
    -webkit-border-radius: 50px;
    border-radius: 50px;
    }

  • 升级一下，画个三角形


    • #triangle-up {
    width: 0;
    height: 0;
    border-left: 50px solid transparent;
    border-right: 50px solid transparent;
    border-bottom: 100px solid red;
    }
    这也就是设置图形的边框做出的三角形，下边框100px厚，左右边框50px不过颜色是透明的就变成了三角形；通过改变各个边框的设置就可以做出其他的三角行了。
